# Build Assets

Place your application icons in this directory:

- `icon.ico` - Windows icon (256x256 minimum, .ico format)
- `icon.png` - macOS/Linux icon (512x512 minimum, .png format)

You can generate icons from a PNG source using online tools like:
- https://icoconvert.com/
- https://www.favicon-generator.org/

For development, placeholder icons will be used automatically.
